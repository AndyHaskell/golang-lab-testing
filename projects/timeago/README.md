# Timeago

## Overview

`timeago` is a Go package for time formating based on current time (eg: "3 hours ago"). Please see the [readme](https://github.com/xeonx/timeago) 
for an example of its usage.

## Step 1: Project Setup

1. :star2: Fork the Github project [https://github.com/xeonx/timeago](https://github.com/xeonx/timeago).

  * :question: If you need help, please see [this document](https://help.github.com/articles/fork-a-repo/) or ask a TA for help.

1. :star2: Clone your fork of your repo to your source path.
